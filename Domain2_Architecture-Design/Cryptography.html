Cryptographic Concepts

A cryptographic algorithm is a set of mathematical functions which uses a (often secret) string of data known as a key, to encrypt/decrypt other data.

    Two main types of encryption algorithms
  • Symmetric Encryption
  • Asymmetric Encryption

  • Hashing algorithms are computed one way - meaning the output cannot be returned to the original plaintext.
    Data States:
  • In-Transit: any data sent across the network
  • At-Rest: Any data currently located in memory, including databases and backups
  • In-Use: Any data actively used by a process
