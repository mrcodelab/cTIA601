Virtualization

Containters vs Virtualization

  • Containers (like Docker) virtualize the OS
  • Virtualization emulates the hardware
    Types of Virtualization:
  • Type 1
    • Runs directly on the system hardware without a host OS
    • Ofent called "native", "bare-metal", or embedded hypervisors
  • Type 2
    • Run within the host OS, acting as a manager of the VM

As A Service

  • IaaS
    • Subscription model whereby the facility is provided and you bring your own hardware
  • PaaS
    • Subscription which allows you to spin many kinds of computing/storage/networking instances (like cloud servers)
  • SaaS
    • Subscription to an application or family of applications
    • No subscriber control over the platform hosting the apps
  • FaaS - Function as a Service
    • Serverless architecture is a software design pattern where applications are hosted by a 3rd service
    • Applications are broken up into functions and can be invoked and scaled individually

Data Loss Prevention

Data Loss Prevention

  • Cloud Based
    • Refers to the moving of administration of DLP capabilities to the cloud
    • Helpful in non-typical environments, especially when remote workers are involved
  • Email
    • DLP solutions can be set to scan emails and attachments, as well as instant messaging services and social media
  • USB Blocking
    • A DLP solution can block USB writes in an environment when the data being written is determined to be confidential
    • Helps prevent data theft via removable media

Backup Strategy

The CompTIA answer is the Full backup + Differential is the fastest because you're only restoring the Full as a baseline and 1 change. Incremental backup represents 1:n.

Recovery Sites

Hot Sites

  • A fully operational facility equipped with all the standard hardware and software used by the organization
  • Most expensive recovery option as they are basically live copies of the production enrivronment

Warm Sites

  • Operationally between a hot and cold site
  • May have hardware and software in a <stopped state, requiring only starting up and a differential backup restore
  • Less expensive than a hot site, but more costly than cold as the system is in near ready condition

Cold Sites

  • A backed up facility that has the necessary infrastructure, but not the actual computer equipment in place
  • A cold site is not ready for immediate operations and typically takes a few days to bring online
  • Least expensive to maintain

Cloud Access Security Broker (CASB)

A {{CASB}} is a tool that monitors traffic as it flows between a cloud service provider and your network. The CASB is used for ensuring that the security policies are followed and enforced, helping prevent things like unencrypted data from being sent to or stored on a cloud solution.

  • Integrates your organization's seucirty-related policies to applications running in the cloud
  • Can detect high-risk applications/users and enforce encryption
