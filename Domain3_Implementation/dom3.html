Access Control Models

  • MAC: Mandatory Access Controls
    • Typical of national defense classified systems
    • Use SELinux to try it out
  • DAC: Discretionary Access Controls
    • Owner-based
  • ABAC: Attribute Based Access Controls
  • RBAC: Role Based Access Controls
    • Think banking systems based on job descriptions
  • Rule-Based Access Control
    • Firewall rules
  • IBAC: Identity Based Access Controls
  • GBAC: Graph-based Access Controls
    • Compared to other approaches like RBAC or ABAC, the main difference is that in GBAC access rights are defined using an organizational query language instead of total enumeration.
